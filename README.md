Описание программы
Графическое приложение для учёта прочитанных книг. Позволяет добавлять книги с указанием названия, автора, жанра и количества страниц. Данные сохраняются в файл books.json. Доступна фильтрация по жанру и количеству страниц. Встроена валидация ввода (проверка на пустые поля и корректность числа страниц). Проект структурирован и готов к работе с Git.

Требования к запуску
Python 3.8+
Библиотека tkinter (входит в стандартную поставку Python).
Для тестов: unittest (входит в стандартную поставку).
Файл данных: books.json (создаётся автоматически).
Инструкция по запуску
Установите Python (если не установлен).
Скачайте или клонируйте этот репозиторий.
Откройте терминал в папке проекта.
Запустите приложение:
python book_tracker.py
Для запуска тестов:
python -m unittest tests.py
Для работы с Git:
Инициализируй репозиторий: git init
Добавь файлы: git add .
Сделай коммит: git commit -m "Initial commit"
Залей на GitHub.
Примеры использования
Добавление книги:
Введите: «Мастер и Маргарита», «М.А. Булгаков», «Роман», «416».
Нажмите «Добавить книгу». Запись появится в таблице.
Фильтрация по жанру:
Выберите жанр «Роман» в выпадающем списке.
В таблице останутся только книги этого жанра.
Фильтрация по страницам:
Введите число «300» в поле фильтра страниц.
В таблице останутся только книги с количеством страниц более 300.
Валидация:
Если оставить поле «Страниц» пустым или ввести текст — появится сообщение об ошибке.
Если ввести отрицательное число — появится сообщение об ошибке.
